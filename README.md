# ecg-heartbeat-classification

# Clasificación multiclase de ECG 

Este proyecto aborda un problema de clasificación de series temporales aplicado al ámbito médico (señales de electrocardiograma). El objetivo es desarrollar un modelo de Deep Learning que pueda clasificar latidos cardíacos en distintas categorías, diferenciando un latido normal de distintos tipos de arritmias.

## Dataset

El dataset no está incluido en el repositorio por limitaciones de tamaño.
Se puede descargar en: https://www.kaggle.com/datasets/shayanfazeli/heartbeat/data

## Ejecución del notebook

Este notebook está optimizado para una ejecución directamente en Google Colab, subiéndolo a través de la pestaña de Github (Upload notebook > Github > pegar el enlace de este repositorio).

Instrucciones para la descarga de los datos: ejecutar las 6 primeras celdas del notebook
1. Cambiar de directorio a /content (celda 1)
2. Eliminar la carpeta del proyecto si existiera (celda 2)
3. Clonar el repositorio (celda 3)
4. Cambiar de directorio (celda 4)
5. Instalar los requirements.txt (celda 5)
6. Descargar el dataset (celda 6)

Se recomienda ejecutar estas primeras celdas individualmente, pero se puede ejecutar directamente todo el notebook con Run all.
